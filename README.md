# 🚀 Align – From Endless Group Chats to Real Plans

> **Align** is a product prototype that helps friend groups make faster and smarter decisions around **time, budget, and preferences** without endless discussions.

![Status](https://img.shields.io/badge/Status-Prototype-6C63FF)
![Built With](https://img.shields.io/badge/Built%20With-HTML%20%7C%20CSS%20%7C%20JavaScript-blue)
![License](https://img.shields.io/badge/License-MIT-green)

## 📖 Problem Statement

Planning with friends usually starts with excitement but often ends with:

- 💬 Hundreds of WhatsApp messages
- ⏳ Endless back-and-forth discussions
- 💸 Hidden budget constraints
- ❌ Last-minute cancellations
- 😓 One organizer doing all the work

Align simplifies this process by collecting everyone's preferences privately and recommending the best decision for the group.

---

# ✨ Features

- 🎯 Create an Event
- 📅 Multi-date voting
- 💰 Private budget constraints
- 🔒 Anonymous preference collection
- 📊 Consensus-based decision visualization
- 📱 Mobile-first responsive UI
- ⚡ Fully interactive prototype
- 🎨 Modern startup-inspired interface

# 🎯 Motivation

Align was built to explore product thinking around collaborative decision-making.

Rather than relying on simple polls, the prototype demonstrates how anonymous constraints, multi-variable preferences, and consensus visualization can help groups make better decisions while reducing organizer effort.
